# Mobile App Assets

This directory should contain the following assets for the Expo app:

## Required Assets

- `icon.png` - App icon (1024x1024px)
- `splash.png` - Splash screen (1242x2436px recommended)
- `adaptive-icon.png` - Android adaptive icon foreground (1024x1024px)
- `favicon.png` - Web favicon (48x48px)
- `notification-icon.png` - Notification icon (96x96px)

## Asset Guidelines

- Use PNG format with transparency where appropriate
- Follow platform-specific guidelines:
  - iOS: Human Interface Guidelines
  - Android: Material Design guidelines
- Ensure assets are optimized for file size
- Test on both light and dark mode if applicable

## Generating Assets

You can use tools like:
- [Expo Asset Generator](https://www.npmjs.com/package/expo-asset-generator)
- [App Icon Generator](https://www.appicon.co/)
- Design tools (Figma, Sketch, etc.)

For MVP, placeholder assets are acceptable, but production apps should have professional, branded assets.
